紧急修复指南 - 编译错误
================================

问题：修改了错误的头文件。有两个SurfaceCardMetadata定义：
1. Engine\Code\Engine\Renderer\Cache\CacheCommon.h (实际使用的)
2. Engine\Code\Engine\Renderer\Cache\SurfaceCacheCommon.h (我错误修改的)

必须修改：CacheCommon.h
================================

修改 1: CacheCommon.h 第 14-38 行的 SurfaceCardMetadata 结构
----------------------------------------------------------------

将：
    float m_originX, m_originY, m_originZ;
    float m_padding0;

    float m_axisXx, m_axisXy, m_axisXz;
    float m_objectID;

    float m_axisYx, m_axisYy, m_axisYz;
    float m_padding2;

    float m_normalX, m_normalY, m_normalZ;
    float m_padding3;

    float m_worldSizeX, m_worldSizeY;
    uint32_t  m_direction;
    uint32_t  m_globalCardID;

改为：
    float m_originX, m_originY, m_originZ;
    float m_objectYaw;           // Object Euler yaw (degrees)

    float m_axisXx, m_axisXy, m_axisXz;
    float m_objectPitch;         // Object Euler pitch (degrees)

    float m_axisYx, m_axisYy, m_axisYz;
    float m_objectRoll;          // Object Euler roll (degrees)

    float m_normalX, m_normalY, m_normalZ;
    float m_objectPosX;          // Object world position X

    float m_worldSizeX, m_worldSizeY;
    float m_objectPosY, m_objectPosZ;  // Object world position Y, Z

    uint32_t  m_direction;
    uint32_t  m_globalCardID;
    uint32_t  m_padding4, m_padding5;  // Align to 16 bytes


修改 2: SceneObject.h - 添加 GetOrientation() getter
----------------------------------------------------------------

在 SceneObject.h 的 public 部分添加：

    const EulerAngles& GetOrientation() const { return m_orientation; }

大约在第 44-46 行附近（GetPosition() 下面）


修改 3: GISystem.cpp - 使用 GetOrientation() 而不是直接访问
----------------------------------------------------------------

在 GISystem.cpp 第 246, 251, 256 行，将：
    obj->m_orientation.m_yawDegrees

改为：
    obj->GetOrientation().m_yawDegrees

同样改 m_pitchDegrees 和 m_rollDegrees


完成这3个修改后，编译应该就能通过了！
